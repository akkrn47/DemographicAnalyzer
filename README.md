# Анализ демографических данных России

## 📋 О проекте
Программа для анализа демографических показателей регионов России с графическим интерфейсом на Qt. Загружает данные из CSV-файла, позволяет фильтровать по регионам и рассчитывать статистические метрики.

## Основные возможности
- Загрузка и парсинг CSV-файлов с демографическими данными
- Отображение таблицы с возможностью фильтрации по региону
- Расчет минимума, максимума и медианы для выбранного демографического показателя
- Валидация входных данных и подсчет некорректных строк
- Автоматическое обновление таблицы при вводе названия региона

## Использование
1. Нажмите "Выберите файл" и укажите CSV-файл с данными
2. Нажмите "Загрузить" для отображения таблицы
3. Введите название региона для фильтрации строк
4. Укажите номер колонки для анализа
5. Нажмите "Рассчитать метрики" для получения статистики

### Ожидаемый формат CSV:
Программа работает с CSV-файлами, содержащими демографические данные в следующем формате:
- Разделитель: запятая (,)
- Обязательные колонки: year, region, npg, birth_rate, death_rate, gdw, urbanization
- Пример файла: russian_demography.csv в репозитории

## Структура программы
- appcontext.h — структуры данных (таблица, метрики, контекст)
- logic.h/cpp — бизнес-логика на C (парсинг CSV, подсчет статистики, управление памятью)
- mainwindow.h/cpp — графический интерфейс на Qt (таблица, кнопки, поля ввода)
- entrypoint.h/cpp — промежуточный слой для вызова операций

## Сборка
- Требуется Qt (версия 5 или 6)
- Откройте.pro файл в Qt Creator
- Соберите проект (Ctrl+B)

## Технологии
- C — обработка данных, парсинг, работа с памятью
- C++/Qt — графический интерфейс
- CSV — формат входных данных
